获取咪咕 userId / token 教程

userId 和 token 只保存在你的 NAS 本机配置里。不要把 token 发给别人，token 相当于你的登录凭据。

1. 打开咪咕官网在电脑浏览器打开咪咕视频官网并登录自己的账号。打开官网
2. 打开开发者工具按键盘 F12。部分笔记本需要同时按 Fn + F12。
3. 切到 Application / 应用Chrome、Edge 浏览器顶部标签里找到 Application；中文界面通常叫“应用”。
4. 找 Cookies左侧展开 Storage / 存储，再展开 Cookies，点击 miguvideo.com 相关域名。
5. 找 userinfo在右侧 Cookie 表格里找到名称为 userinfo 的这一行。
6. 复制 userinfo 的值复制 Value / 值这一整段。它通常由“用户 ID + token”组成。
7. 填到本软件设置里把前半部分填到 userId，后半部分填到 token。填写后点保存。

画质说明：不填写只能标清；填写后可用高清；蓝光、原画、4K 需要咪咕 VIP 账号并受频道权益限制。

TVBox 使用教程

TVBox 下载地址建议下载开源版、官方原版。https://github.com/vipshihua/tvbox打开
1. 数据源地址填 TVBox 配置 JSON。
2. 直播源地址填直播频道列表，推荐 TXT。
3. 直播 EPG 地址填节目单地址。

如果 TVBox 只让填一个地址，就只填第 1 个数据源地址；如果有三行，就按上面三项分别填写。
